This project focuses on **Heterogeneous Ensemble Learning**, where different types of algorithms collaborate to make the final prediction.

# 🤖 What is Ensemble Learning?

Ensemble Learning is a machine learning technique that combines predictions from multiple models to produce a stronger and more reliable model.

The key idea is:

> **Many weak or diverse models together often outperform a single model.**

---

# 🧠 What is Heterogeneous Ensemble?

A Heterogeneous Ensemble consists of **different machine learning algorithms** working together.

Example:

- Decision Tree 🌳
- Logistic Regression 📈
- Support Vector Machine ⚡
- Random Forest 🌲
- K-Nearest Neighbors 📍

Each model has different strengths and weaknesses.

Combining them helps reduce:

- Variance
- Bias
- Overfitting
